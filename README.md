# Белякова Кира ИТ-1-2024, Лабораторная работа №4, 2 вариант
## Задание
Реализовать класс Money для работы с денежными суммами (поля: рубли — unsigned int, копейки — short int). В классе должны быть предусмотрены: конструкторы, свойства для доступа к полям, перегрузка оператора вывода и метод добавления произвольного количества копеек к объекту. Дополнительно реализовать унарные операции (++ и --) для копеек, явное приведение к unsigned int (рубли) и неявное к double (копейки в долях рубля), а также бинарные операции сложения и вычитания копеек.
## Алгоритм решения
1. Проектирование структуры класса: описываю приватные поля rubles и kopeks для обеспечения инкапсуляции.
2. Реализация конструкторов: создаю конструктор по умолчанию, конструктор с параметрами и конструктор копирования.
3. Создание метода нормализации: пишу внутреннюю функцию, которая переводит каждые 100 копеек в 1 рубль и следит, чтобы количество копеек не было отрицательным или избыточным.
4. Добавление копеек: реализую метод, который прибавляет unsigned int к полю копеек и вызывает нормализацию.
5. Перегрузка вывода: перегружаю оператор << для форматированного отображения денежной суммы.
6. Реализация унарных операций: перегружаю ++ и -- для изменения значения копеек на единицу.
7. Реализация приведения типов: прописываю оператор explicit operator unsigned int() для получения целой части (рублей) и operator double() для получения дробной части (копеек).
8. Реализация бинарных операций: перегружаю + и - для взаимодействия объекта Money с беззнаковым целым числом (копейками), добавив проверку, чтобы сумма не стала меньше нуля.
